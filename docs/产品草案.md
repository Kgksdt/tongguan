# 产品草案

## 项目名称

通关

## 核心目标

把算法学习从“我感觉学过”变成“我知道哪里没过关、下一步练什么、什么时候算真正会了”。

## 第一阶段功能

- 首页仪表盘：显示当前阶段、已通关知识点、本周任务、复习提醒。
- 知识地图：按 CSP-S、提高、CF1600、CF1800、CF2000 分层展示知识点。
- 通关标准：每个知识点都有概念、模板、变式、讲解、复盘五类标准。
- 题单记录：记录题目来源、难度、结果、耗时、错因和复习日期。
- 错题复盘：按错因分类，形成后续复习队列。

## 第二阶段功能

- 教师模式：把某个知识点生成课堂讲解、例题、常见错误和课后作业。
- 评价模板：保存常用学生评价模板。
- 自动反馈：根据标注生成学生版、家长版、机构版评价。

## 本地优先原则

- 优先本地运行。
- 初期用浏览器 localStorage 保存。
- 数据稳定后再考虑导入导出 JSON、Excel 或 SQLite。

## 通关状态

- 未开始
- 学习中
- 练习中
- 待复盘
- 已通关

## 单个知识点通关标准

- 能说清核心思想。
- 能默写标准模板。
- 能独立 AC 3-5 道代表题。
- 能讲给学生听。
- 能隔几天重做错题。
